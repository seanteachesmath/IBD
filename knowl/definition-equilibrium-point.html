Definition 2.1.1. Equilibrium point.

An equilibrium point is a value of the solution that is unchanging in time. For a differential equation
\begin{equation*} \frac{d^{} {x}}{d {t}^{}}=f(x), \end{equation*}
\(x^*\) is an equilibrium point if \(f(x^*)=0\text{.}\) Stability of equilibria helps us characterize the long term behavior of a model using information from equilibria. While there are mathematical varieties of stability we will consider local asymptotic stability.
