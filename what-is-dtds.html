    Section 3.1 What is a discrete-time dynamical system?

    A discrete-time dynamical system (DTDS), also known as a recurrence relation or difference equation, is a way of generating a sequence of numbers by repeatedly applying a calculation. The mechanics are similar to the way we might calculate the Fibonacci or Pingala numbers starting from \(F_0 = 0\) and \(F_1 = 1\) and continuing for \(n \geq 2\) with
    \begin{equation*} F_{n} = F_{n-1} + F_{n-2}\text{.} \end{equation*}
    Though this was an early example of a mathematical model, and is well-described elsewhere, we will now look in other directions that have more biological relevance.

    Subsection 3.1.1 A linear DTDS

    Suppose the amount of a radioactive medical tracer decays by \(11\%\) per hour (in other words \(89\%\) persists into the next hour). If we take \(x_{t}\) to be the amount at hour \(t\text{,}\) the DTDS is given by
    \begin{equation*} x_{t+1} = 0.89x_{t} \end{equation*}
    starting from the initial amount \(x_{0}\text{.}\) This means \(x_{1} = 0.89 x_{0}\) and \(x_{2} = 0.89 x_{1}\text{.}\) Since we already know \(x_{1}\) in terms of \(x_0\text{,}\) we can write \(x_{2} = (0.89)^2x_{0}\text{.}\) This pattern extends to a solution of the form
    \begin{equation*} x_{t} = (0.89)^{t}x_{0}\text{.} \end{equation*}
    In terms of \(t\)-dependence, this is an exponential function, the form is a constant to a variable power, but in discrete time we call this pattern geometric decay (or growth).
    By defining a function \(x(t)\) that is exact with \(x_{t}\) at integer \(t\)-values and interpolates between them, we can establish an important connection between discrete and continuous dynamics. Noticing that we can write \(0.89 = e^{\ln(0.89)}\text{,}\) then raising each side to the power \(t\text{,}\) we have
    \begin{equation*} (0.89)^{t} = e^{\ln(0.89)t}\text{.} \end{equation*}
    Now multiplying each side by the initial condition \(x_{0}\text{,}\) we have that \(x_{t} = (0.89)^{t}x_{0}\) Similarly, \(x(t) = x_{0}e^{\ln(0.89)t}\text{.}\) These are identical at integer values of \(t\) and the exponential smoothly interpolates the geometric at times in between the integer values. Additionally this gives a connection between the discrete growth factor, here \(0.89\text{,}\) and the exponential (decay) rate \(\ln(0.89) \approx -0.1165\text{.}\)

    Subsection 3.1.2 The logistic map

    A very famous model is called the logistic map, given by
    \begin{equation*} x_{t+1} = rx_{t}(1-x_{t})\text{.} \end{equation*}
    This somewhat parallels the continuous logistic growth differential equation in its structure. This is more explicit if we instead write
    \begin{equation*} N_{t+1} = rN_{t}\left(1-\dfrac{N_{t}}{K}\right)\text{.} \end{equation*}
    Then we can let \(x_{t} = \dfrac{N_{t}}{K}\) or \(N_{t} = Kx_{t}\text{.}\) Substituting we have \(Kx_{t+1} = rKx_{t}(1-x_{t})\) and after dividing the common \(K\) we have
    \begin{equation*} x_{t+1} = rx_{t}(1-x_{t})\text{,} \end{equation*}
    which matches the more common version stated earlier.

    Example 3.1.2. The parameter \(r\).

    The model is defined for \(0 \leq x_{t} \leq 1\text{.}\) To ensure that \(0 \leq x_{t+1} \leq 1\) we have to be careful with the choice of \(r\text{.}\) Use calculus to find the maximum \(r\)-value to ensure that the function doesn’t exceed one in value. You are welcome to write the function as \(f(x) = rx(1-x)\) prior to beginning your analysis.

Following from the initial value, each time we hit the updating function on the cobwebbing diagram, we append a new value to our solution.
