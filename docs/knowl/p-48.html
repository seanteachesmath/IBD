The equilibrium is found from \(\dfrac{dN}{dt} = 0\) or setting and solving \(0 = rN^* + a\text{.}\) Hence, the equilibrium is \(N^* = -\dfrac{a}{r}\text{.}\) Now, this only makes sense when \(r<0\text{,}\) otherwise \(N^*\) itself is negative. If \(r>0\) the population grows naturally, aided more so by immigration.
