Solution 2.1.7.1.

Eigenvalues are solutions to
\begin{equation*} r^2+3r+2=0, \end{equation*}
or which gives solutions \(r_+ = -1\) and \(r_- = -2\text{.}\) Substituting the eigenvalues one at a time, we can solve for entries in each eigenvector.
First, for \(r_+ = 5\) we have
\begin{equation*} \begin{pmatrix} (1 - (-1))\amp -2\\ 3 \amp (-4 -(-1)) \end{pmatrix} \cdot \begin{pmatrix}u_+\\v_+\end{pmatrix} = 0 \end{equation*}
which corresponds to the system of equations
\begin{equation*} \begin{cases} 2 u_+ + -2v_+ \amp = 0\\ 3 u_+ -3 v_+ \amp = 0 \end{cases} \end{equation*}
The setup is very similar for the second eigenvalue, and solving these both we have \(r_+ = -1\) with \(u_+ = 1\) and \(v_+ = 1\) and \(r_- = -2\) with \(u_- = 2\) and \(v_- = 3\text{.}\)
