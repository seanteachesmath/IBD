Looking at the first equation for \(n=0\text{,}\)
\begin{equation*} \frac{d^{} {P_0(t)}}{d {t}^{}}=(-1)b P_{-1}(t) +(1)\mu P_{1}(t)-0\cdot (b -\mu) P_{0}(t), \end{equation*}
we simplify to
\begin{equation*} \frac{d^{} {P_0(t)}}{d {t}^{}}= \mu P_{1}(t). \end{equation*}
If we were to look at the equilibrium of \(P_0\text{,}\) assuming \(\mu\neq0\text{,}\) we would need \(P_1(t)\rightarrow0\text{.}\) Since \(P_1\) appears in the next equation, we require all \(P_i(t)\rightarrow0\) for \(i>0\text{.}\) Since \(P_i(t)\) are probabilities, we require \(\displaystyle\sum_0^\infty P_i(t)=1\text{,}\) which means that \(P_0(t)\rightarrow1\text{.}\) This suggests that extinction is unavoidable, but a bit more work provides additional insight. Specifically populations with \(b>m\) can escape extinction, with the probability of extinction of
\begin{equation*} \left(\dfrac{m}{b}\right)^{N_0} \end{equation*}
which for \(b>m\) decays geometrically as a function of the initial population size. With quite a bit of effort, this result is very nicely illustrated in classic texts.
