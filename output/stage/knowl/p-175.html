For a concrete example, use the matrix \(A\text{,}\) we’ve introduced earlier. For this, \(\operatorname{tr}(A) = 11\) and \(\operatorname{det}(A) = 24\text{.}\) The equation for the eigenvalues is
\begin{equation*} 0 = \lambda^2 - 11\lambda + 24. \end{equation*}
We could use the quadratic formula, and more generally we should, but for a quick start, this one factors to
\begin{equation*} 0 = (\lambda - 3)(\lambda-8). \end{equation*}
The eigenvalues are \(\lambda = 3, 8\text{.}\) To remind you,
\begin{equation*} \lambda_{\pm} = \frac{-(-11)\pm\sqrt{(-11)^2 - 4(1)(24)}}{2(1)}, \end{equation*}
it’s important to watch out for the sign of the \emph{discriminant} \((-11)^2 - 4(1)(24)\text{,}\) which, if negative, would cause complex-valued solutions.
