A very famous model is called the logistic map, given by
\begin{equation*} x_{t+1} = rx_{t}(1-x_{t})\text{.} \end{equation*}
This somewhat parallels the continuous logistic growth differential equation in its structure. This is more explicit if we instead write
\begin{equation*} N_{t+1} = rN_{t}\left(1-\dfrac{N_{t}}{K}\right)\text{.} \end{equation*}
Then we can let \(x_{t} = \dfrac{N_{t}}{K}\) or \(N_{t} = Kx_{t}\text{.}\) Substituting we have \(Kx_{t+1} = rKx_{t}(1-x_{t})\) and after dividing the common \(K\) we have
\begin{equation*} x_{t+1} = rx_{t}(1-x_{t})\text{,} \end{equation*}
which matches the more common version stated earlier.
