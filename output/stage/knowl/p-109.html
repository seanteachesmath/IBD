First, for \(r_+ = i\) we have
\begin{equation*} \begin{pmatrix} (1 - i)\amp 2\\ -1 \amp (-1 -i) \end{pmatrix} \cdot \begin{pmatrix}u_+\\v_+\end{pmatrix} = 0 \end{equation*}
which corresponds to the system of equations
\begin{equation*} \begin{cases} 2 u_+ + -2v_+ \amp = 0\\ 3 u_+ -3 v_+ \amp = 0 \end{cases} \end{equation*}
The setup is very similar for the second eigenvalue, and solving these both we have \(r_+ = -1\) with \(u_+ = -1-i\) and \(v_+ = 1\) and \(r_- = -i\) with \(u_- = -1+i\) and \(v_- = 1\text{.}\)
