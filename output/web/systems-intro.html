    Section 2.1 Introduction to 2D systems

    To achieve our goals, a little bit from the theory of differential equations is helpful. Ultimately we will linearize a nonlinear system near equilibrium using the Jacobian matrix.
    Consider the generic 2D linear system
    \begin{equation*} \begin{cases} \dfrac{dx}{dt} \amp = f(x, y)\\ \dfrac{dg}{dt} \amp = g(x, y) \end{cases} \end{equation*}
    For the choice \(f(x, y) = ax+by\) and \(g(x,y) = cx+dy\) (sorry about using the letter \(d\text{,}\) but it becomes standard in the next step), we can establish some important information about solutions. Much of this is linked to properties of
    \begin{equation*} A = \left(\begin{array}{cc}a\amp b\\c\amp d\end{array}\right)\text{.} \end{equation*}
    Two important properties of this matrix are the trace
    \begin{equation*} \tau = \operatorname{Tr}(A) = a + d, \end{equation*}
    and the determinant
    \begin{equation*} \Delta = \operatorname{Det}(A) = ad-bc. \end{equation*}
    We begin by assuming that solutions to the system are exponential, for example,
    \begin{equation*} \begin{pmatrix}x(t)\\y(t)\end{pmatrix} = \begin{pmatrix}u\\v\end{pmatrix}\cdot e^{rt}, \end{equation*}
    for some vector of unknown constants
    \begin{equation*} \begin{pmatrix}u\\v\end{pmatrix}. \end{equation*}
    Now we differentiate and substitute,
    \begin{equation*} \dfrac{d}{dt}\left(\begin{pmatrix}x\\y\end{pmatrix}\right) = \begin{pmatrix}u\\v\end{pmatrix}\cdot re^{rt}. \end{equation*}
    Equating these,
    \begin{equation*} \begin{pmatrix}u\\v\end{pmatrix}\cdot re^{rt} = \begin{pmatrix}a\amp b\\c\amp d\end{pmatrix}\cdot \begin{pmatrix}u\\v\end{pmatrix}\cdot e^{rt} \end{equation*}
    Moving all terms to one side,
    \begin{equation*} 0 = \begin{pmatrix}a\amp b\\c\amp d\end{pmatrix}\cdot \begin{pmatrix}u\\v\end{pmatrix}\cdot e^{rt} - \begin{pmatrix}u\\v\end{pmatrix}\cdot re^{rt}. \end{equation*}
    Or,
    \begin{equation*} 0 = \left(\begin{pmatrix}a\amp b\\c\amp d\end{pmatrix} - rI\right)\begin{pmatrix}u\\v\end{pmatrix}\cdot e^{rt}, \end{equation*}
    where \(I\) is the \(2\times2\) identity matrix.
    The exponential can never be zero, so we are looking for solutions to
    \begin{equation*} 0 = \left(\begin{pmatrix}a\amp b\\c\amp d\end{pmatrix} - rI\right)\begin{pmatrix}u\\v\end{pmatrix}\text{,} \end{equation*}
    which are given by the eigenvector and eigenvalue of the matrix \(A\text{.}\) Eigenvalues are given by solutions to \(\operatorname{det}(A-rI)=0\) or
    \begin{equation*} \operatorname{det}\begin{pmatrix}a-r\amp b\\c\amp d-r\end{pmatrix} = 0. \end{equation*}
    We solve \((a-r)(d-r)-bc=0\) for \(r\text{.}\)
    The equation can be written \(r^2 - (a + d)r + ad-bc = 0\) and solved for \(r\) and then the eigenvector. Notice that the equation can be rewritten in terms of the matrix properties referenced above,
    \begin{equation*} r^2 - \tau r + \Delta = 0. \end{equation*}
    The trace and determinant show up once again. By the quadratic formula
    \begin{equation*} r = \dfrac{\tau \pm \sqrt{(-\tau)^2-4\Delta}}{2}. \end{equation*}
    Frome here a lot could happen, depending on the entries of the matrix and the resulting trace and determinant. Roots, \(r\text{,}\) could be
    • Real and distinct, opposite sign
    • Real and distinct, same sign
    • Real and repeated
    • Complex
    • Purely imaginary,
    roughly in the order you might expect them based on past experience with the quadratic formula. What follows is the most economical summary I can provide of one of my favorite chapters in ordinary differential equations.

    Subsection 2.1.1 Real and distinct eigenvalues of opposite sign

    As we will see in this section, this situation corresponds to phase plane dynamics corresponding to a saddle at the equilibrium.
    Real and distinct roots is probably the simplest case to introduce. Let’s call those solutions \(r_+\) and \(r_-\text{,}\) eventually we will need numbers, but first it’s worth pointing out how we find the corresponding eigenvectors. We find those by solving
    \begin{equation*} 0 = \begin{pmatrix}a - r_\pm\amp b\\c\amp d - r_\pm\end{pmatrix}\begin{pmatrix}u_\pm\\v_\pm\end{pmatrix}. \end{equation*}
    The subscripts indicate that in the case of real and distinct roots we will have one solution for each solution from the quadratic formula.

    Example 2.1.1.

    Find nullclines, equilibria, and sketch sample solutions for the linear system given by
    \begin{equation*} \begin{cases} \dfrac{dx}{dt} \amp = x+2y\\ \dfrac{dy}{dt} \amp = 4x+3y \end{cases} \end{equation*}

    Example 2.1.2. Real and distinct eigenvalues of opposite sign.

    Consider the matrix
    \begin{equation*} \begin{pmatrix}1\amp 2\\4\amp 3\end{pmatrix}\text{.} \end{equation*}
    The trace is \(\tau = 1 + 3 = 4\) and the determinant is \(\Delta = (1)(3)-(2)(4) = -5\text{.}\) Find both eigenvalues and corresponding eigenvectors.
    Solution.
    Eigenvalues are solutions to
    \begin{equation*} r^2-4r-5=0, \end{equation*}
    or
    \begin{equation*} r_\pm = \dfrac{4\pm\sqrt{16-4(-5)}}{2} = \dfrac{4\pm\sqrt{36}}{2} = 2\pm3 = 5, -1. \end{equation*}
    Substituting the eigenvalues one at a time, we can solve for entries in each eigenvector.
    First, for \(r_+ = 5\) we have
    \begin{equation*} \begin{pmatrix}(1 - 5)\amp 2\\3\amp (4 - 5)\end{pmatrix}\cdot \begin{pmatrix}u_+\\v_+\end{pmatrix} = 0 \end{equation*}
    which corresponds to the system of equations
    \begin{equation*} \begin{cases} -4 u_+ + 2v_+ \amp = 0\\ 3 u_+ -1 v_+ \amp = 0\\ \end{cases} \end{equation*}
    The setup is very similar for the second eigenvalue, and solving these both we have \(r_+ = 5\) with \(u_+ = 1\) and \(v_+ = 2\) and \(r_- = -1\) with \(u_- = -1\) and \(v_- = 1\text{.}\)
    In the phase plane, the eigenvectors represent what are called straight-line solutions.
    The conclusion to Example 2.1.2 means that for the linear system of differential equations ... we can write solutions for any initial condition in terms of unknowns \(c_+ \) and \(c_-\) as
    \begin{equation*} \begin{pmatrix}x(t)\\y(t)\end{pmatrix} = c_+\begin{pmatrix}u_+\\v_+\end{pmatrix}e^{r_{+}t} + c_-\begin{pmatrix}u_-\\v_-\end{pmatrix}e^{r_{-}t} \end{equation*}
    Some solutions will start on the first eigenvalue (this means an initial condition \((x_0,y_0)\) whose values are some constant multiple of those of the first eigenvector.) The contribution of the second eigenvector will be zero, and solutions will grow exponentially away from the origin along the first eigenvector. Others will start on the second eigenvalue (this means an initial condition \((x_0,y_0)\) whose values are some constant multiple of those of the second eigenvector.) The contribution of the first eigenvector will be zero, and solutions will decay exponentially towards the the origin along the second eigenvector. For all other initial conditions we have a contribution from both, but with the contribution of the first eigenvector growing and the contribution of the second decaying exponentially over time. We call this dynamic in the phase plane a saddle.
    For example, with the initial condition \(x_0 = 3\) and \(y_0=6\text{,}\) we have \(c_+ = 3\) and \(c_- = 0\text{.}\) The solution grows exponentially in proportion to the vector.

    Example 2.1.3.

    Use the trace and determinant of each matrix to find eigenvalues and eigenvectors.
    • \(\displaystyle A = \begin{pmatrix} 2\amp4\\3\amp1\end{pmatrix}\)
    • \(\displaystyle A = \begin{pmatrix} 1\amp2\\3\amp4\end{pmatrix}\)

    Subsection 2.1.2 Real and distinct eigenvalues of the same sign

    As we will see in this section, this situation corresponds to phase plane dynamics corresponding to a node at the equilibrium. Solutions will grow away from equilibrium if both roots are of positie sign and grow towards the equilibrium if both roots are of negative sign. This is due to the exponential nature of the solutions.

    Example 2.1.4.

    Find nullclines, equilibria, and sketch sample solutions for the linear system given by
    \begin{equation*} \begin{cases} \dfrac{dx}{dt} \amp = x-2y\\ \dfrac{dy}{dt} \amp = 4x-3y \end{cases}. \end{equation*}

    Example 2.1.5. Real and distinct eigenvalues of same sign.

    Consider the matrix
    \begin{equation*} \begin{pmatrix}1\amp -2\\ 3\amp -4\end{pmatrix}\text{.} \end{equation*}
    The trace is \(\tau = 1 - 4 = -3\) and the determinant is \(\Delta = (1)(-4)-(-2)(3) = 2\text{.}\) Find both eigenvalues and corresponding eigenvectors.
    Solution.
    Eigenvalues are solutions to
    \begin{equation*} r^2+3r+2=0, \end{equation*}
    or which gives solutions \(r_+ = -1\) and \(r_- = -2\text{.}\) Substituting the eigenvalues one at a time, we can solve for entries in each eigenvector.
    First, for \(r_+ = 5\) we have
    \begin{equation*} \begin{pmatrix} (1 - (-1))\amp -2\\ 3 \amp (-4 -(-1)) \end{pmatrix} \cdot \begin{pmatrix}u_+\\v_+\end{pmatrix} = 0 \end{equation*}
    which corresponds to the system of equations
    \begin{equation*} \begin{cases} 2 u_+ + -2v_+ \amp = 0\\ 3 u_+ -3 v_+ \amp = 0 \end{cases} \end{equation*}
    The setup is very similar for the second eigenvalue, and solving these both we have \(r_+ = -1\) with \(u_+ = 1\) and \(v_+ = 1\) and \(r_- = -2\) with \(u_- = 2\) and \(v_- = 3\text{.}\)
    The conclusion to Example 2.1.5 means that with the initial condition \(x_0 = 3\) and \(y_0=3\text{,}\) we have \(c_+ = 3\) and \(c_- = 0\text{.}\) The solution grows exponentially from the equilibrium at the origin along the vector. With the initial condition \(x_0 = 4\) and \(y_0=6\text{,}\) we have \(c_+ = 0\) and \(c_- = 2\text{.}\) The solution decays exponentially towards the equilibrium at the origin along the vector.

    Subsection 2.1.3 Real and repeated eigenvalues

    This happens to be a borderline case that takes a bit of work, but fortunately doesn’t play a major role in most applications. From the solution to the quadratic, this happens when \(0 = (-\tau)^2 - 4\Delta\) or, solving for \(\Delta\text{,}\)
    \begin{equation*} \Delta = \frac{\tau^2}{4}. \end{equation*}
    This is along the curve that separates real and complex solutions.

    Subsection 2.1.4 Complex eigenvalues

    By the quadratic formula, any complex roots must be distinct, but some may be purely imaginary with no real part. Recall, we are trying to solve
    \begin{equation*} r^2 - \tau r + \Delta = 0 \end{equation*}
    for \(r\text{.}\) We are interested in cases where \((-\tau)^2-4\Delta\) from the quadratic formula is negative and the square root imaginary.
    \begin{equation*} r = \dfrac{\tau \pm \sqrt{(-\tau)^2-4\Delta}}{2}. \end{equation*}
    Since our main goal is to assess local stability of eqiulibria in nonlinear models by linearizing, we won’t go so far here as writing explicit general solutions to the linear problems.

    Example 2.1.7.

    Find nullclines, equilibria, and sketch sample solutions for the linear system given by
    \begin{equation*} \begin{cases} \dfrac{dx}{dt} \amp = x+2y\\ \dfrac{dy}{dt} \amp = -x-y \end{cases}. \end{equation*}

    Example 2.1.8. Real and distinct eigenvalues of same sign.

    Consider the matrix
    \begin{equation*} \begin{pmatrix}1\amp 2\\ -1\amp -1\end{pmatrix}\text{.} \end{equation*}
    The trace is \(\tau = 1 - 1 = 0\) and the determinant is \(\Delta = (1)(-1)-(2)(-1) = 1\text{.}\) Find both eigenvalues and corresponding eigenvectors.
    Solution.
    Eigenvalues are solutions to
    \begin{equation*} r^2+1=0, \end{equation*}
    or which gives solutions \(r_+ = i\) and \(r_- = -i\text{.}\) Substituting the eigenvalues one at a time, we can solve for entries in each eigenvector.
    First, for \(r_+ = i\) we have
    \begin{equation*} \begin{pmatrix} (1 - i)\amp 2\\ -1 \amp (-1 -i) \end{pmatrix} \cdot \begin{pmatrix}u_+\\v_+\end{pmatrix} = 0 \end{equation*}
    which corresponds to the system of equations
    \begin{equation*} \begin{cases} 2 u_+ + -2v_+ \amp = 0\\ 3 u_+ -3 v_+ \amp = 0 \end{cases} \end{equation*}
    The setup is very similar for the second eigenvalue, and solving these both we have \(r_+ = -1\) with \(u_+ = -1-i\) and \(v_+ = 1\) and \(r_- = -i\) with \(u_- = -1+i\) and \(v_- = 1\text{.}\)
    The conclusion to Example 2.1.8 might be surprising. If you’re aware of the complex plane, you might think we can graph these. Unfortunately the complex plane has a real axis and an imaginary axis. Or phase plane has two real axes corresponding to model variables. So, as it is, we can’t visualize complex eigenvalues in the same way as real ones.
    This means that we no longer start with real-valued solutions. Recall that we guessed that \(e^{rt}\text{,}\) was a part of our solution, but now we have complex-valued exponentials since \(r = \lambda + \mu i\) for real numbers \(\lambda\) and \(\mu\text{.}\) Taking just this part, we have
    \begin{equation*} e^{(\lambda + \mu i)t} = e^{\lambda t}\cdot e^{\mu i t}\text{.} \end{equation*}
    By Euler’s formula, we can write these as real-valued solutions that involve products of trigonometric and exponential functions. Solutions will be purely trigonometric (closed periodic orbits) if the real part of the eigenvalues is zero, and otherwise be damped or growing oscillations in the phase plane if the real part is negative or positive, respectively. The trigonometric part comes from writing
    \begin{equation*} e^{\mu i t} = \cos(\mu t) + i\sin(\mu t) \end{equation*}
    using Euler’s formula. Since our goal here is just to lern about the linearized system, it won’t be especially helpful to pursue analytical solutions any longer at this point.
