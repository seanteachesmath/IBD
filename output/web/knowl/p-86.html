Real and distinct roots is probably the simplest case to introduce. Let’s call those solutions \(r_+\) and \(r_-\text{,}\) eventually we will need numbers, but first it’s worth pointing out how we find the corresponding eigenvectors. We find those by solving
\begin{equation*} 0 = \begin{pmatrix}a - r_\pm\amp b\\c\amp d - r_\pm\end{pmatrix}\begin{pmatrix}u_\pm\\v_\pm\end{pmatrix}. \end{equation*}
The subscripts indicate that in the case of real and distinct roots we will have one solution for each solution from the quadratic formula.

The per predator feeding rate is,
\begin{equation*} \frac{N_C}{T}=\left(\frac{\hat h N}{1+\hat s \hat h N}\right), \end{equation*}
since the definition of \(T\) was ‘per predator’. Redefining constants \(h=\hat h\) and \(s=\hat s \hat h\text{,}\) we can finally write the per predator harvesting rate as
\begin{equation*} \frac{N_C}{T}=h_{II}(N)=\left(\frac{h N}{1+ s N}\right), \end{equation*}
so that the harvesting term is \(h_{II}(N)F.\) If predators instantaneously consume their prey, \(s=0\) which returns the Type I response, because saturation vanishes.
