At \((B^*, C^*) = \left(\dfrac{m}{\epsilon p}, \dfrac{r}{p}\left(1-\dfrac{m}{\epsilon p K}\right)\right)\text{,}\) which is positive provided \(K\gt \frac{m}{\epsilon p}\text{.}\) Under this condition we have
\begin{equation*} J(B^*, C^*) = \begin{pmatrix} -\dfrac{rm}{\epsilon p K} \amp -\dfrac{m}{\epsilon}\\r\left(\epsilon - \dfrac{m}{p K}\right) \amp 0\end{pmatrix} \end{equation*}
for which \(\tau = -\dfrac{rm}{\epsilon p K}\) and \(\Delta = rm\left(1-\dfrac{m}{\epsilon p K}\right)\text{.}\) The trace is negative, so the nontrivial equilibrium will be stable provided the determinant is positive. The determinant is positive provided \(K\gt \frac{m}{\epsilon p}\text{,}\) which is the condition under which this equilibrium is positive.
