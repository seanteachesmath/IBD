For this model, the Jacobian is
\begin{equation*} J(B, C) = \begin{pmatrix} r - \frac{2rB}{K} - p C \amp -p B\\\epsilon p C \amp \epsilon p B - m\end{pmatrix}\text{.} \end{equation*}
At each equilibrium,
  • At \((B^*, C^*) = (0, 0)\) we have
    \begin{equation*} J(0, 0) = \begin{pmatrix} r \amp 0\\0 \amp - m\end{pmatrix}\text{,} \end{equation*}
    for which \(\tau = r-m\) and \(\Delta = -rm\text{.}\) Since the determinant is negative, the trace is irrelevant and the equilibrium is a saddle.
  • At \((B^*, C^*) = (K, 0)\) we have
    \begin{equation*} J(K, 0) = \begin{pmatrix} -r \amp -p K\\0 \amp \epsilon p K - m\end{pmatrix}\text{,} \end{equation*}
    for which \(\tau = -r + \epsilon p K - m\) and \(\Delta = -r(\epsilon p K - m)\text{.}\) This is a saddle if \(\epsilon p K - m \gt 0\) or \(K \gt \dfrac{m}{\epsilon p}\text{.}\) Otherwise, if \(\epsilon p K - m \lt 0\) or \(K \lt \dfrac{m}{\epsilon p}\text{,}\) the determinant is positive and the trace is negative, indicating a stable node or spiral. Since this equilibrium is on the horizontal axis, reality suggests that it must be a stable node, not spiral - otherwise solutions would take on positive and negative values to spiral towards the equilibrium. Additionally, solutions could not cross the horizontal axis (a \(C\)-nullcline) with vertical motion.
  • At \((B^*, C^*) = \left(\dfrac{m}{\epsilon p}, \dfrac{r}{p}\left(1-\dfrac{m}{\epsilon p K}\right)\right)\text{,}\) which is positive provided \(K\gt \frac{m}{\epsilon p}\text{.}\) Under this condition we have
    \begin{equation*} J(B^*, C^*) = \begin{pmatrix} -\dfrac{rm}{\epsilon p K} \amp -\dfrac{m}{\epsilon}\\r\left(\epsilon - \dfrac{m}{p K}\right) \amp 0\end{pmatrix} \end{equation*}
    for which \(\tau = -\dfrac{rm}{\epsilon p K}\) and \(\Delta = rm\left(1-\dfrac{m}{\epsilon p K}\right)\text{.}\) The trace is negative, so the nontrivial equilibrium will be stable provided the determinant is positive. The determinant is positive provided \(K\gt \frac{m}{\epsilon p}\text{,}\) which is the condition under which this equilibrium is positive. Interestingly this equilibrium can be a stable node or a stable spiral (or the degenerate in-between case) depending on the value of \(r\text{.}\) We can carefully inspect the quantity
    \begin{align*} \tau^2-4\Delta \amp = \left(-\dfrac{rm}{\epsilon p K}\right)^2 - 4rm\left(1-\dfrac{m}{\epsilon p K}\right)\\ \amp = \dfrac{rm}{\epsilon p K}\left(\dfrac{rm}{\epsilon p K} - 4\left(\epsilon p K-m\right)\right)\text{.} \end{align*}
    This will lead to a stable spiral if
    \begin{equation*} \dfrac{rm}{\epsilon p K} - 4\left(\epsilon p K - m\right) \lt 0\text{.} \end{equation*}
    This involves many of the parameters weâ€™ve focused on so far, but now also \(r\text{.}\) In particular,
    \begin{equation*} \dfrac{rm}{\epsilon p K} - 4\left(\epsilon p K - m\right)\lt 0 \end{equation*}
    if
    \begin{equation*} r \lt 4\epsilon p K\left(\frac{\epsilon p K}{m} - 1\right)\text{.} \end{equation*}
    If \(r\) exceeds this, the dynamics will be a stable node (no oscillations). Keep in mind that due to earlier assumptions of this case (the existence of the equilibrium itself, it is given that \(K\gt \frac{m}{\epsilon p}\)).
