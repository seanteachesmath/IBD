Consider the generic 2D linear system
\begin{equation*} \begin{cases} \dfrac{dx}{dt} \amp = f(x, y)\\ \dfrac{dg}{dt} \amp = g(x, y) \end{cases} \end{equation*}
For the choice \(f(x, y) = ax+by\) and \(g(x,y) = cx+dy\) (sorry about using the letter \(d\text{,}\) but it becomes standard in the next step), we can establish some important information about solutions. Much of this is linked to properties of
\begin{equation*} A = \left(\begin{array}{cc}a\amp b\\c\amp d\end{array}\right)\text{.} \end{equation*}
Two important properties of this matrix are the trace
\begin{equation*} \tau = \operatorname{Tr}(A) = a + d, \end{equation*}
and the determinant
\begin{equation*} \Delta = \operatorname{Det}(A) = ad-bc. \end{equation*}
