Consider the generic 2D linear system
\begin{equation*} \begin{cases} \dfrac{dx}{dt} \amp = f(x, y)\\ \dfrac{dy}{dt} \amp = g(x, y) \end{cases} \end{equation*}
For the choice \(f(x, y) = ax+by\) and \(g(x,y) = cx+dy\) (sorry about using the letter \(d\text{,}\) but it becomes standard in the next step), it looks like
\begin{equation*} \begin{cases} \dfrac{dx}{dt} \amp = ax+by\\ \dfrac{dy}{dt} \amp = cx+dy \end{cases}\text{.} \end{equation*}
It is convenient to express this in matrix-vector form,
\begin{equation*} \begin{pmatrix}\frac{dx}{dt}\\\frac{dy}{dt}\end{pmatrix} = \begin{pmatrix}a\amp b\\c\amp d\end{pmatrix}\cdot \begin{pmatrix}x\\y\end{pmatrix}. \end{equation*}
