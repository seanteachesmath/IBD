We begin by assuming that solutions to the system are exponential, for example,
\begin{equation*} \begin{pmatrix}x(t)\\y(t)\end{pmatrix} = \begin{pmatrix}u\\v\end{pmatrix}\cdot e^{rt}, \end{equation*}
for some vector of unknown constants
\begin{equation*} \begin{pmatrix}u\\v\end{pmatrix}. \end{equation*}
Now we differentiate and substitute,
\begin{equation*} \dfrac{d}{dt}\left(\begin{pmatrix}x\\y\end{pmatrix}\right) = \begin{pmatrix}u\\v\end{pmatrix}\cdot re^{rt}. \end{equation*}
Equating these,
\begin{equation*} \begin{pmatrix}u\\v\end{pmatrix}\cdot re^{rt} = \begin{pmatrix}a\amp b\\c\amp d\end{pmatrix}\cdot \begin{pmatrix}u\\v\end{pmatrix}\cdot e^{rt} \end{equation*}
Moving all terms to one side,
\begin{equation*} 0 = \begin{pmatrix}a\amp b\\c\amp d\end{pmatrix}\cdot \begin{pmatrix}u\\v\end{pmatrix}\cdot e^{rt} - \begin{pmatrix}u\\v\end{pmatrix}\cdot re^{rt}. \end{equation*}
Or,
\begin{equation*} 0 = \left(\begin{pmatrix}a\amp b\\c\amp d\end{pmatrix} - rI\right)\begin{pmatrix}u\\v\end{pmatrix}\cdot e^{rt}, \end{equation*}
where \(I\) is the \(2\times2\) identity matrix.
